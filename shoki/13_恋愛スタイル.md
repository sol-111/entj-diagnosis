# 恋愛スタイル：ENTJ男が好む恋愛の形

> 恋愛の各フェーズで主導する認知機能が入れ替わるため、外から見ると「一貫性がない」ように見える。実際は極めて構造的。フェーズ別の攻略図。

## 恋愛の5フェーズ

### PHASE 1：追撃期（Te＋Se全開）

「この人だ」の直感で行動が爆速化。店選び・計画・連絡の全主導権を握り、火星山羊座10Hの「結果が出るまで止まらない」が発動。普段の回避型が一時停止する瞬間最大風速。相手が簡単に靡かないほど燃える。

### PHASE 2：査定期（Te＋Ni稼働）

付き合うか否かの判断フェーズ。知性・価値観・将来性を静かにスキャンし「この関係に意味があるか」を問い続ける。相手からは「急に何を考えてるか分からなくなった」と見えるタイミング。ここは冷めたのではなく、審査中。

### PHASE 3：構築期（Pragma全開）

「本質的に合う」が確定すると一気にコミットが深まる。感情の告白より、関係の設計——会う頻度・役割・共通の目標——を整備していく。恋愛がプロジェクト化する。

### PHASE 4：安定期（Ni＋Te未来設計）

「5年後どうなっているか」が次の判断軸。MC射手座が「人生の方向性は同じか」を問う。方向が割れるとここで急速に冷え、揃えば関係は「同志」に進化する。

### PHASE 5：深化期（月蠍座8H発動）

表面は安定、内側で最深の絆が形成される。「自分にだけ見せる顔」の相互開示、秘密の共有——ここで初めてFi劣等が緩む。この段階まで来た関係の終わりは魂に傷を残す。裏切りへの反応も最大化する。

## 連絡・距離感の仕様書

### LINEの理想運用

- 基本は**1日1〜3往復の「内容がある」やり取り**。ニュースの共有、進捗、相談——用事ベースが心地いい
- 「元気？」だけの定期便は不要どころか負荷。質＞量
- 既読スルーは相互に無罪。「既読無視で不機嫌になる」は最大級の相性不一致シグナル

### 会う頻度

- 付き合いたて週1〜2、安定期は週1または隔週でも成立する
- 「会ってない＝冷めてる」ではない。**会った時の密度で勝負する**型

### 一人時間 — 譲れない一線

一人っ子として「自分だけの時間と空間」が生活のデフォルトだった。他者と空間を共有し続けること自体がエネルギーを消費する——これはワガママではなく構造。**「一緒にいても別々のことをしている時間」を愛せる関係**、並走型の関係が理想形。ここが理解されないと関係は致命傷を負う。

## 喧嘩のパターンと修復プロトコル

### パターン1：正論の連射 → 逃げ場のない裁判

Te×タイプ1の「正しさ」が止まらず、相手を論理で追い詰める。本人は事実確認のつもり、相手には人格審問。
**修復：** 相手が黙り始めたら勝ちではなく敗戦処理の開始。一旦引いて時間を置く。「だって事実じゃん」の追加投入は全損コース。

### パターン2：静かなスイッチオフ → 突然の撤退

怒鳴る代わりに消えるのがこの男の喧嘩。批判増→口数減→連絡途絶の3段階（タイプ4化）。
**修復：**「批判が増えた」段階での対話が唯一の早期解決ルート。撤退開始後は追わず、「○日に話そう」の枠だけ置く。本人側の宿題は「引く前に一言」。

### パターン3：Pragma的冷却 → 意味の消失

停滞・退屈・方向のズレが蓄積すると、「この関係に意味がない」の判定が下り、冷徹なほど合理的に撤退が始まる。
**修復：** 共通の目標の更新が唯一のリセットボタン。新しい共同プロジェクト・旅・挑戦。「変わらなくていいじゃん」は最後の一押しになるので厳禁。

## 結婚観 — 人生の共同プロジェクト

太陽7H×MC射手座×タイプ1が描く結婚は、生活共同体ではなく**「何を成し遂げるかを共に描くパートナーシップ」**。対等で、互いのキャリアを尊重し、相乗効果を生む関係。

- **同棲の必須要件：** 各自のスペース（書斎・自室）。「自分だけの領域」は精神安定の根幹
- **家事：**「察して」ではなく明文化された合理的分担
- **食事：** 平日は各自でも可、週末は一緒に。共有カレンダーで予定は可視化
- **子育て：** 持つなら本気で設計する（タイプ1）。「なんとなく」では持たない

### 関係維持の5原則

1. 停滞させない——共通の目標を更新し続ける
2. 距離感を更新する——フェーズごとに最適距離は変わる
3. 日常の誠実さを積む——金星乙女座6Hの信頼形成
4. 深化のタイミングを逃さない——月蠍8Hの絆を定期確認
5. 互いの変化を楽しむ——固定より進化
